Como Calcular Média, Moda e Mediana

Neste artigo, vamos explorar três importantes medidas estatísticas: média, moda e mediana. Estas medidas são fundamentais na análise de conjuntos de dados para resumir suas características principais e entender a distribuição dos valores.

A média é a soma de todos os valores dividida pelo número de elementos. É utilizada para obter uma representação central dos dados, sendo sensível a valores extremos.

A moda é o valor mais frequente em um conjunto de dados. É útil para identificar valores mais comuns ou populares em uma distribuição.

A mediana é o valor que divide um conjunto de dados ordenado ao meio. Ela é menos sensível a valores extremos do que a média e é útil para entender a distribuição simétrica dos dados.

Média

A média é calculada somando todos os valores de um conjunto de dados e dividindo pelo número total de valores.

Fórmula da média:

 Média = (soma dos valores) / (número de valores)

Exemplo:

 Dados: 3, 5, 7, 10, 15
 Média = (3 + 5 + 7 + 10 + 15) / 5
 Média = 8

Moda

A moda é o valor que mais se repete em um conjunto de dados.

Exemplo:

 Dados: 2, 3, 3, 5, 5, 5, 7
 Moda = 5 (porque 5 é o valor que mais se repete)

Mediana

A mediana é o valor que divide o conjunto de dados em duas metades iguais, quando os dados estão ordenados.

Exemplo:

 Dados: 2, 4, 6, 8, 10
 Mediana = 6 (porque é o valor do meio)

Quando o conjunto de dados tem um número par de elementos, a mediana é a média dos dois valores do meio.

Exemplo:

 Dados: 1, 3, 5, 7
 Mediana = (3 + 5) / 2
 Mediana = 4

Exercícios

Tente resolver os seguintes exercícios:

  1. Calcule a média dos números: 10, 15, 20, 25, 30.
  2. Encontre a moda dos números: 2, 4, 4, 6, 6, 6, 8.
  3. Calcule a mediana dos números: 5, 10, 15, 20, 25, 30.

Respostas:

  1. Média: 20
  2. Moda: 6
  3. Mediana: 17.5
